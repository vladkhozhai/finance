- **Total Balance Card**: `bg-gradient-to-br from-blue-50 to-indigo-50`

- **Stale Rate Card**: Same as default
- **Error Card**: `bg-red-50` with `border-red-200`

- **Headings**: `text-zinc-900`

- **Card Left Border**: Custom from payment method color

## Interaction States

### Hover
- **Payment Method Card**: `hover:shadow-md` transition
- **Transaction Row**: `hover:bg-zinc-50`
- **Button**: Color darkens

### Click
- **Payment Method Card**: Opens transaction list below
- **Close Button**: Dismisses transaction list
- **Pagination**: Changes page

### Focus (Keyboard Navigation)
- **Card**: Focus ring appears
- **Button**: Focus ring with offset
- **Tooltip Trigger**: Keyboard accessible
